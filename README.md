## ✨ 核心特性 (Features)

WebDavClient 不仅仅是一个文件管理器，它是一个追求极致交互体验的**私有云中枢**。

### 🎨 极致 UI 设计
*   **无边框悬浮岛 (Floating Island)**：侧边栏采用大圆角悬浮设计，配合磨砂玻璃背景，营造出悬浮于内容之上的高级感。
*   **灵动岛操作栏 (Dynamic Action Bar)**：底部的功能区随选中状态智能变形，将常用操作（重命名、删除、移动）汇聚指尖。
*   **自适应布局**：
    *   **桌面端**：悬浮侧边栏 + 高效多列网格。
    *   **移动端**：沉浸式抽屉导航 + 双列精致卡片。
*   **中英双语**：内置国际化支持，一键无缝切换语言。

### 🚀 强大的文件管理
*   **多网盘聚合**：
    *   **本地存储**：直接管理服务器/电脑的主目录文件 (`~/`)，实时监控磁盘容量。
    *   **WebDAV 挂载**：支持添加坚果云、Alist、Nextcloud 等任意 WebDAV 协议网盘，并在同一界面无缝切换。
*   **全能预览**：无需下载，直接在浏览器中预览图片、视频（流式播放）、音频、PDF 及代码文本。
*   **智能拖拽**：
    *   **上传**：拖拽文件到浏览器即可极速上传。
    *   **下载**：支持将网页中的文件直接拖拽到电脑桌面（Chrome/Edge）。
    *   **整理**：拖拽文件到文件夹即可完成移动。
*   **高效操作**：支持批量删除、剪切/粘贴（跨网盘移动）、重命名、新建文件夹。
*   **搜索与排序**：支持实时文件名模糊搜索，以及按名称、日期、类型排序。

## 🛠️ 技术栈 (Tech Stack)

*   **前端**：React 18, Vite, Tailwind CSS, Framer Motion (动画), React Dropzone.
*   **后端**：Node.js, Express, WebDAV Client (代理转发), FS-Extra.
*   **架构**：前后端分离但同构部署，Node.js 作为反向代理服务器解决跨域与认证问题。

## ⚡ 快速开始 (Getting Started)

### 前置要求
*   Node.js (v16 或更高版本)

### 1. 安装依赖
```bash
# 一键安装根目录及前后端所有依赖
npm run install-all
```

### 2. 启动服务
```bash
# 同时启动前端 (Vite) 和后端 (Express)
npm run dev
```

访问浏览器：`http://localhost:5173`

*后端服务默认运行在 8000 端口，前端代理已自动配置。*

## 📖 使用指南

1.  **添加网盘**：
    *   点击侧边栏底部的 **"添加 WebDAV 网盘"**。
    *   输入名称、WebDAV 地址 (如 `https://dav.jianguoyun.com/dav/`)、账号密码。
    *   点击 **"测试连接"** 确保畅通，然后保存。
2.  **文件操作**：
    *   **多选**：点击文件右上角的圆圈（网格模式）或复选框（列表模式）。移动端支持长按选中。
    *   **重命名**：选中单个文件后，底部灵动岛会出现 "重命名" 按钮。
    *   **移动文件**：选中文件 -> 点击 "移动" -> 进入目标文件夹 -> 点击 "粘贴到此处"。
3.  **视图切换**：点击右上角的图标，可在 **网格视图** (Grid) 和 **列表视图** (List) 间切换。

## 🛡️ 安全说明

*   **隐私优先**：所有 WebDAV 凭据仅存储在你本地服务器的 `server/drives.json` 文件中，绝不上传云端。
*   **路径保护**：后端内置了路径遍历防御机制，防止访问未授权的系统文件。
*   **无感认证**：后端代理了所有 WebDAV 请求，前端预览时无需重复输入密码，且解决了浏览器的跨域限制。
